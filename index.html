작동 방식

산모수첩 사진 한 장이면 됩니다.

  1. 1

    사진 업로드

    산모수첩·진료기록을 찍어 올리면 AI가 문서 종류를 판별하고 핵심 정보를 추출합니다. (못 읽은 항목은 직접 입력)

  2. 2

    회복 프로파일로 정리

    흩어진 정보가 한 장으로 — 출산방식·경과일·통증·발열·오로·수유·EPDS·지역.

  3. 3

    오늘의 안내

    가능/금지 운동 + 위험 신호 체크 + 근거 칩을 매일 안내합니다.

안전 우선 설계

AI는 읽고·정리·연결만 합니다.
결정은 규칙이 합니다.

AI가 하는 것
  • 문서 읽기 (OCR)
  • 비정형 정보 구조화
  • 영상 분류·매칭
  • 자연어 요약
AI가 하지 않는 것
  • 진단
  • 처방
  • 응급 여부 결정
  • 의학적 권고 직접 생성
판단 주체 = 가이드라인 기반 Rule Engine. 레드플래그 감지 시 운동·분석은 전면 차단됩니다.

“AI가 못 하는 게 아니라, 안전을 위해 의도적으로 안 합니다.”
